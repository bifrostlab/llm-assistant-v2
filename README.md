# LLM Assistant Discord Bot

Porting [llm-assistant](https://github.com/bifrostlab/llm-assistant) into TS. Project is bootstrapped with [ts-starter-template](https://github.com/samhwang/ts-starter-template)


## Available scripts

```shell
pnpm run start
pnpm run test
pnpm run format
pnpm run build
```
